Named Functions
---------------
    - The function with the particular "user defined name"
      called as Named Function.

Syntax:
-------
    //function definition
    function functionname(parameters with datatype)
                                        :returntype{
        //business logic
    }

    //call the function 
    functionname(arguments);

Ex:
---
    function fun_one():string{
        retutn "welcome to ES6 Functions";
    };    
    fun_one();
